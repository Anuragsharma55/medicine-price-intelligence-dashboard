# medicine-price-intelligence-dashboard

An interactive Streamlit dashboard that compares medicine prices across online pharmacy sellers using Google Shopping data.

The system extracts real-time product data using SerpAPI and provides price comparison, visual analytics, and best-price identification to help users make informed purchasing decisions.

## Features

- Real-time medicine price comparison
- Automatic lowest-price detection
- Interactive visualizations
- Seller-wise price analysis
- Direct purchase links
- Streamlit dashboard interface

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- SerpAPI (Google Shopping API)

---

## How It Works

1. User enters a medicine name.
2. The system fetches product results from Google Shopping.
3. Prices across sellers are extracted and cleaned.
4. The dashboard compares vendors and identifies the lowest-cost option.
5. Visualizations show price distribution across sellers.
